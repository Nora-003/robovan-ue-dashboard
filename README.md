# Robovan UE 交互模型

Robovan 单位经济模型（Unit Economics）可视化仪表盘，支持三种商业模式的实时对比与参数调优。

## 在线访问

https://nora-003.github.io/robovan-ue-dashboard

## 功能特性

- **三模式对比**：平台模式 / 租赁模式 / 卖车模式
- **实时联动**：修改左侧参数，右侧图表与指标即时刷新
- **月度 + 年度双视角**：单月成本结构拆解 + 全年跨模式对比
- **敏感性分析**：各关键因子 ±10% 对单车毛利的影响
- **计算公式展示**：关键成本口径透明可查

## 本地使用

直接用浏览器打开 `index.html` 即可，无需构建工具。

## 技术栈

- React 18 (CDN)
- Recharts (CDN)
- Tailwind CSS (CDN)
- Babel Standalone (CDN)
